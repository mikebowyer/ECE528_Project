This API allows interaction with the Crowdsourced Data project. The base URL for this project is https://dy0duracgd.execute-api.us-east-1.amazonaws.com/dev. All methods are defined relative to this base URL.

  • /share-image
    Upload an image with required metadata to the Crowdsourced database.
    • Latitude: float64
      • Latitude (degrees) where the image was taken.
    • Longitude: float64
      • Longitude (degrees) where the image was taken.
    • ImageBase64: string
      • Image bytes encoded as a base64 string.
Response will be given in JSON format: 
{
    "statusCode": 200,
    "message": "Success",
    "dynamoMeta":
    {
        "Latitude": 20.2,
        "Longitude": 30.3,
        "EpochTime": 1617556321,
        "ImageURL": "https://ktopolovbucket.s3.amazonaws.com/original_1617556321.jpg",
        "LabeledImageURL": "https://ktopolovbucket.s3.amazonaws.com/labeled_1617556321.jpg",
        "humanReadableTime": "2021-04-04 17:12:01",
        "Labels": ["Road", "Traffic Jam"]
    }
}
  • /get-imgs-in-gps-box
    Retrieve all images in database within a given latitude-longitude box.
    • BR_Long: float64
      • Bottom right longitude (degrees) for bounding box.
    • BR_Lat: float64
      • Bottom right latitude (degrees) for bounding box.
    • TL_Long: float64
      • Top left longitude (degrees) for bounding box.
    • TL_Lat: float64
      • Top left latitude (degrees) for bounding box.
Response will be given in JSON format: 
{
    "statusCode": 200,
    "message": "RecievedMessage: {"TL_Lat": "0.0", "TL_Long": "0.0", "BR_Lat": "180.0", "BR_Long": "180.0"}",
    "body": [
       {
           "info": {
               "labeled_image_source":"https://ktopolovbucket.s3.amazonaws.com/labeled_1617556321.jpg",
               "detected_labels": ["Road", "Traffic Jam",]
               "human_readable_time": "2021-04-04 17:12:01",
               "latitude": 20.2,
               "longitude": 30.3,
               "image_source": "https://ktopolovbucket.s3.amazonaws.com/original_1617556321.jpg"
           }
       }
    ]
}
