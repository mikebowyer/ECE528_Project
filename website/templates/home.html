Distributed Driving Data Collection Through The Cloud

Overview

This website allows you to visualize data that is uploaded through the API to assist in distributed collection of driving footage. The website shows a markers indicating where data was uploaded. When selected, the image captured, latitude, longitude, and objects detected will be displayed.
